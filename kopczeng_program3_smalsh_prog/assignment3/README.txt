This program can be compiled on the os1 server by typing the following:
gcc -o smallsh main.c

To then run the program, type the following:
./smallsh

If you want to use the test script we were provided, type one of the following:
./p3testscript 2>&1
./p3testscript 2>&1 | more
./p3testscript > mytestresults 2>&1
